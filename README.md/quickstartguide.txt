## Troubleshooting

**Server won't start?**
```bash
# Check if port 3000 is in use
# Windows:
netstat -ano | findstr :3000

# Mac/Linux:
lsof -i :3000
```

**Database issues?**
```bash
# Delete and recreate
rm backend/suraksha360.db
npm start
```

**Location not working?**
- Allow location permission in browser
- Must use HTTPS in production
