## ğŸ”§ TECHNICAL STACK

### Backend:
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** SQLite3
- **Authentication:** JWT (JSON Web Tokens)
- **Password Hashing:** bcryptjs
- **File Upload:** Multer
- **CORS:** Enabled for development

### Frontend:
- **Citizen App:** Vanilla HTML/CSS/JavaScript
- **Admin Dashboard:** HTML/CSS/JavaScript + Chart.js
- **Charts:** Chart.js for analytics visualization
- **Storage:** localStorage for offline support
- **Geolocation:** HTML5 Geolocation API
- **Media:** MediaRecorder API for audio/video

### Database Schema:
- users (Citizens + Admins)
- crime_reports (Anonymous submissions)
- sos_alerts (Emergency alerts)
- evidence (Audio/video files)
- risk_zones (High-risk area definitions)
- sms_logs (SMS notification tracking)

## ğŸŒ API ENDPOINTS

### Authentication
- POST `/api/auth/send-otp` - Send Aadhaar OTP (mocked)
- POST `/api/auth/verify-otp` - Verify OTP
- POST `/api/auth/register` - Register new citizen
- POST `/api/auth/admin-login` - Admin login

### SOS Emergency
- POST `/api/sos/trigger` - Trigger emergency alert
- POST `/api/sos/evidence/:sosId` - Upload evidence
- GET `/api/sos/active` - Get active alerts (admin only)
- GET `/api/sos/all` - Get all alerts (admin only)
- GET `/api/sos/:sosId` - Get SOS details
- PUT `/api/sos/:sosId/resolve` - Resolve alert
- GET `/api/sos/evidence/file/:evidenceId` - Download evidence

### Crime Reports
- POST `/api/reports/submit` - Submit anonymous report
- GET `/api/reports/all` - Get all reports (admin only)
- GET `/api/reports/:reportId` - Get report details
- GET `/api/reports/voice/:reportId` - Get voice recording
- POST `/api/reports/sync` - Sync offline reports

### Risk Zones
- POST `/api/riskzones/create` - Create risk zone (admin only)
- GET `/api/riskzones/all` - Get all risk zones
- PUT `/api/riskzones/:zoneId` - Update risk zone
- DELETE `/api/riskzones/:zoneId` - Delete risk zone
- POST `/api/riskzones/check` - Check if location is in risk zone

### Analytics
- GET `/api/analytics/stats` - Dashboard statistics
- GET `/api/analytics/crime-types` - Crime type distribution
- GET `/api/analytics/area-density` - Area-wise crime density
- GET `/api/analytics/time-trends` - Time-based crime trends
- GET `/api/analytics/sos-response` - SOS response time analysis
- GET `/api/analytics/risk-zone-effectiveness` - Risk zone effectiveness
